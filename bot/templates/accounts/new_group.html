Инструкция

Обязательно выполните эти действия:

(Все действия делать через компьютер или версию сайта для ПК с телефона)

  • Открыть вкладку «Управление» на странице сообщества
  • Во вкладке «Сообщения»:
    1. Включить сообщения сообщества и нажать на кнопку «Сохранить»
  • Во вкладке «Настройки для бота»
    1. Включить возможности ботов
    2. Включить пункт «Разрешать добавлять сообщество в беседы»
    3. нажать на кнопку «Сохранить»
  • Перейти в раздел «Настройки»
  • Перейти в подраздел «Работа с API»
  • Перейти на вкладку «Long Poll API»
  • На вкладке «Настройки»:
    1. Включить Long Poll API
    2. Выбрать версию API 5.120 или новее
  • На вкладке «Типы событий»:
    1. Включить «Входящее сообщение»
  • Перейти на вкладку «Ключи доступа»:
    1. Нажать на кнопку«Создать ключ»
    2. Разрешить доступ ко всем пунктам
    3. Нажать на кнопку «Создать»
    4. Подтвердить номер телефона
    5. Скопировать ключ доступа и вставить в поле «Ключ доступа» вверху этой страницы
  • Теперь получаем id сообщества:
    1. Вернуться на страницу сообщества
    2. Скопировать ссылку на сообщество
    3. Перейти на сайт REGVK
    4. Вставить ссылку на сообщество и нажать на кнопку «Определить ID»
    5. Скопировать id сообщества из пункта «ID публичной страницы»
    6. Вставить id в поле «id сообщества» вверху этой страницы
  • Нажать кнопку «Войти» вверху этой страницы
